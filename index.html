Range Map

Zu meinen Projekten gehörte auch die Entwicklung einer Range-Map – eine interaktive Karte, auf der Sie verschiedene Flugzeugmodelle auswählen können. Dabei können Sie den Startpunkt (Flughafen) festlegen, und die Karte wird automatisch neu generiert, wobei die neuen Koordinaten und die Reichweite jedes ausgewählten Modells eingezeichnet werden. Diese innovative Lösung ermöglicht es Ihnen, auf einen Blick zu sehen, wie weit Sie mit jedem Modell fliegen können.

Webshop - dotCoffee

Ich bin besonders stolz auf meinen Webshop, den ich im Rahmen meines FH-Projekts entwickelt habe. Es handelt sich um ein lokales Projekt, bei dem Gäste, User und Admins verschiedene Benutzerrechte haben. Gäste können Produkte anschauen, User können sie in den Warenkorb legen, und Admins haben Zugriff auf ein Dashboard zur Verwaltung von Produkten und User Accounts. Für das Backend nutzten wir Java, Spring Boot, Hibernate und MySQL, während wir im Frontend JavaScript, HTML, CSS und Bootstrap einsetzten.
